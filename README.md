# Voting App

Create, share, vote.

### Installation

#### `Step 1` - clone the repo
  
```bash
$ git clone https://github.com/jtCodes/voting-app
```

#### `Step 2` - cd in the repo

```bash
$ cd voting-app
```

#### `Step 3` - install dependencies

```bash
$ npm install
```

#### `Step 4` - run application

```bash
$ npm run start
```

In browser, open [http://localhost:3000](http://localhost:3000)
